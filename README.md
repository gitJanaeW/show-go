# SHOWGO

## Purpose

 Help users find multiple new music events with the help of they're Spotify account genre preferences in any location of the users choosing!

## Built With

- HTML
- CSS
- JavaScript
- Bulma
- Moment.js
- Spotify server side API
- Ticketmaster server side API
